To run the game, simply run the following command in the terminal:

> python game.py

To navigate between each board, use the green buttons to move the current board in the selected direction.
To rotate a row/ column, select the 'Toggle Controls' button and use the red buttons to move the corresponding row/ column in the selected direction.

Enjoy!
